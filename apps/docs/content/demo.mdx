# Demo Project

To see AdmiNext in action, check out demo version:

[AdmiNext Demo](https://github.com/max-konin/AdmiNext/tree/master/apps/demo)

This demo provides a hands-on example of how AdmiNext works, including:

- Setting up and managing resources
- Handling CRUD operations with different backends
- Customizing the Admin Panel
- Configuring routes, dashboards, and custom pages

Currently, the demo includes an integration with Prisma ORM and Drizzle ORM, showcasing how to fetch and manage data.
