# Testing

AdmiNext provides a package with utilities to help you test your application. The package is called `@adminext/playwright`.
It contains a set of page objects to help you create e2e tests with [playwright](https://playwright.dev/).

To install the package run:

```sh npm2yarn copy
npm i  @adminext/playwright
```

The package contains 3 page objects:

- ResourceListViewPage
- ResourceEditPage
- ResourceNewPage

You can check examples of how to use them in the [demo project](https://github.com/max-konin/AdmiNext/tree/master/apps/demo/tests/prisma).
